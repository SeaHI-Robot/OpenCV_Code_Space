# README

>  OpenCV-CPP Template & Elegent CMake Configuration



#### Description
我在Ubuntu中的使用VSCode写CPP-OpenCV的工作空间；

此工作空间配置较为优雅方便管理；
- `src` 目录放置 cpp 源代码
- make 之后 `bin` 目录存放编译好的二进制可执行文件
- `build` 目录存放 CMake 相关文件和 Makefile；

#### Usage

- `mkdir build`
- `cd build`
- `cmake ..`
- `make`
